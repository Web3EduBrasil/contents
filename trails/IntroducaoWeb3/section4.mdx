# **Introdução a Blockchain**

## **O que é Blockchain e como Funciona:**

- Blockchain é uma tecnologia de banco de dados distribuído que armazena informações em blocos conectados em uma cadeia. Cada bloco contém um registro de transações e é interligado ao bloco anterior por meio de criptografia.

- Cada vez que um novo bloco é adicionado, ele é verificado por uma rede de computadores. Esse processo é conhecido como consenso, onde a maioria dos participantes precisa validar as informações para garantir a segurança e autenticidade.

## **Propriedades do Blockchain:**

- **Descentralizado:** Os dados não ficam armazenados em um único local; cada participante da rede possui uma cópia.

- **Imutável:** Uma vez que um dado é registrado em um bloco, ele não pode ser alterado. Isso garante que os registros são confiáveis.

- **Transparente e Acessível:** Qualquer pessoa pode verificar as transações, mas apenas o proprietário tem a chave para acessar seus dados.

## **Tipos de Blockchain:**

- **Blockchain Pública:** Qualquer pessoa pode participar da rede e verificar as transações, como ocorre no Bitcoin e Ethereum.

- **Blockchain Privada:** É restrito a um grupo específico de participantes, sendo mais usado por empresas para armazenar dados internos.

- **Blockchain Híbrida:** Combinação de características públicas e privadas, ideal para redes que buscam transparência, mas com controle sobre certos dados.

## **Mecanismos de Consenso:**

- **Prova de Trabalho (Proof of Work):** Computadores competem para resolver problemas complexos; é o método usado pelo Bitcoin.

- **Prova de Participação (Proof of Stake):** Usuários são escolhidos para validar transações com base na quantidade de tokens que possuem. Ethereum migrou para esse modelo recentemente para consumir menos energia.
